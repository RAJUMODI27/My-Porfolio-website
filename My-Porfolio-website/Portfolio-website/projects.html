My Projects

Mp3 PLayer
"Royplayer" is a simple music player GUI application built using Python's Tkinter and Pygame libraries. It allows users to play, pause, stop, and navigate through a playlist of MP3 songs. The interface features a dynamic status bar, a song position slider, and basic playback controls, offering an intuitive and engaging music listening experience. The application also leverages the Mutagen library to display song information and calculate song lengths accurately.

Weather App
The script utilizes Python's requests, json, and pyttsx3 libraries to fetch and announce the current temperature of a specified city using a free weather API. The user is prompted to enter the city name, and the script fetches the current temperature from the Weather API. The pyttsx3 library is used to convert the temperature information into speech, providing an audible response to the user. Note: For the code to work, you need to replace "YOUR_API_KEY" in the API URL with your actual weather API key.

Jarvis Virtual AI
"Jarvis" is a multifunctional virtual assistant project created using various Python libraries. This voice-activated assistant is equipped to perform a range of tasks, including providing the current time, sharing jokes, capturing screenshots, and implementing voice authentication. Leveraging libraries such as pyttsx3 for text-to-speech functionality and integrating diverse features, Jarvis offers an interactive and personalized user experience. With its diverse capabilities, Jarvis showcases the potential of Python in developing a dynamic and engaging virtual assistant.

Lanuage Translater
This Voice Translation AI, built with Python, employs speech recognition to understand user input, translates it into the selected language, and then audibly communicates the translation using either the playsound or pyttsx3 library. Users can choose from various output languages for a personalized translation experience.
